## 90% of my git workflow

```bash
git add -u

git commit -m 'message'

git push origin master
```

## gitignore

```
#  set a global gitignore
git config --global core.excludesfile '~/.gitignore'

#  show global gitignore
git config --get core.excludesfile
```
